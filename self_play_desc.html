Instructions

On the left of the screen is the pattern area. Here you will see multiple grids where the pattern is applied.

On the right of the screen is the description area. Here is where you will describe the pattern you see.

We have broken up the description into 3 areas to make sure that you include enough information. These 3 description categories are:

  • What You See: Here, you should describe what you should see on the input grid.
  • What You Do: Here, you should describe what you do to transform the input grid into the output grid. This is the pattern.
  • What You See: Here, if the grid size changes, you should describe how it changed.

Make sure you use enough detail so that you can create the output based on only the description and an input grid that you have not seen before.
